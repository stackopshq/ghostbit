REST API

Base URL: {{ base_url }}/api/v1

All encryption is client-side (AES-256-GCM). The server only stores ciphertext — it can never read your data. Interactive docs: /docs

POST /api/v1/pastes

Create a new end-to-end encrypted paste. The content field must be Base64-encoded AES-256-GCM ciphertext produced client-side.

Request body (JSON)
{
  "content":     "string  (required — base64 AES-256-GCM ciphertext)",
  "nonce":       "string  (required — base64 12-byte GCM nonce)",
  "kdf_salt":    "string  (optional — base64 PBKDF2 salt, for password pastes)",
  "language":    "string  (optional, e.g. python)",
  "expires_in":  "integer (optional, TTL in seconds, ≥ 1)",
  "burn":        "boolean (optional, default false)",
  "max_views":   "integer (optional, ≥ 1)",
  "webhook_url": "string  (optional — URL to POST on each view)"
}
Response 201
{
  "id":           "abc123",
  "url":          "{{ base_url }}/abc123",
  "delete_token": "xxxxxxxxxxxxxxxxxxxx",
  "expires_at":   1234567890,
  "burn":         false,
  "max_views":    null
}
Example
curl -X POST {{ base_url }}/api/v1/pastes \
  -H "Content-Type: application/json" \
  -d '{"content": "<base64_ciphertext>", "nonce": "<base64_nonce>", "language": "python"}'
GET /api/v1/pastes/{id}

Retrieve ciphertext and metadata. The caller must decrypt client-side using the key from the URL #fragment. This counts as a view.

Response 200
{
  "id":           "abc123",
  "content":      "<base64 ciphertext>",
  "nonce":        "<base64 nonce>",
  "kdf_salt":     "<base64 salt> | null",
  "language":     "python",
  "created_at":   1234567890,
  "expires_at":   1234567890,
  "burn":         false,
  "max_views":    5,
  "view_count":   1,
  "has_password": false
}
Example
curl {{ base_url }}/api/v1/pastes/abc123
DELETE /api/v1/pastes/{id}

Permanently delete a paste. Requires the X-Delete-Token header returned at creation.

Headers (required)
X-Delete-Token: xxxxxxxxxxxxxxxxxxxx
Example
curl -X DELETE {{ base_url }}/api/v1/pastes/abc123 \
  -H "X-Delete-Token: xxxxxxxxxxxxxxxxxxxx"
POST /api/v1/detect

Detect the programming language of a plaintext snippet.

Request body (JSON)
{ "content": "def hello():\n    print('world')" }
Response 200
{ "language": "python" }

Error codes

400Content too large, invalid webhook URL, or invalid input
403Invalid delete token
404Paste not found or expired
429Rate limit exceeded
