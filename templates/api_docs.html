REST API

Base URL: {{ base_url }}/api/v1

POST /api/v1/pastes

Create a new paste. Returns the paste ID, URL, and a delete token.

Request body (JSON)
{
  "content":    "string  (required)",
  "language":   "string  (optional, e.g. python)",
  "expires_in": "integer (optional, seconds — e.g. 86400 for 1 day)",
  "burn":       "boolean (optional, default false)",
  "max_views":  "integer (optional, minimum 1)",
  "password":   "string  (optional)"
}
Response 201
{
  "id":           "abc123",
  "url":          "{{ base_url }}/abc123",
  "delete_token": "xxxxxxxxxxxxxxxxxxxx",
  "expires_at":   1234567890,
  "burn":         false,
  "max_views":    null
}
Example
curl -X POST {{ base_url }}/api/v1/pastes \
  -H "Content-Type: application/json" \
  -d '{"content": "print(\"hello\")", "language": "python", "expires_in": 86400}'
GET /api/v1/pastes/{id}

Retrieve a paste. For password-protected pastes, pass the password via the X-Password header.

Headers (optional)
X-Password: your_password
Response 200
{
  "id":         "abc123",
  "content":    "print(\"hello\")",
  "language":   "python",
  "created_at": 1234567890,
  "expires_at": 1234567890,
  "burn":       false,
  "max_views":  5,
  "view_count": 1
}
Example
curl {{ base_url }}/api/v1/pastes/abc123
DELETE /api/v1/pastes/{id}

Delete a paste. Requires the X-Delete-Token header returned at creation.

Headers (required)
X-Delete-Token: xxxxxxxxxxxxxxxxxxxx
Example
curl -X DELETE {{ base_url }}/api/v1/pastes/abc123 \
  -H "X-Delete-Token: xxxxxxxxxxxxxxxxxxxx"

Error codes

400Content too large or invalid input
401Password required (missing X-Password header)
403Incorrect password or invalid delete token
404Paste not found or expired
